Requirements:
pip install streamlit
pip install google-generativeai
pip install fitz
pip install python-dotenv
pip install Pillow
pip install pandas
pip install frontend
pip install tools

Create a static folder.
Create a file named .env in the project root directory. Add this in it: GOOGLE_API_KEY=<your-google-api-key>. 

Run the application using 
streamlit run app.py.
